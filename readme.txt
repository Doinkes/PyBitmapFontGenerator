PyBitmapFontGenerator
Прграмма для генерации bitmap текстуры шрифта. Особенность в том, что все символы не смотря на их
размер помещаются в свою клетку. Доступен только выбор шрифта из вне, доступен выбор цвета текста,
можно менять параметры матрицы и рамер шрифта.
Для работы с текстовым полем символов (выделение, копирование, вставка), используйте английскую раскладку клавиатуры.

Используйте встроенное виртуальное окружение.
Компиляция программы в 1 exe без питона в системе: pyinstaller --onefile --windowed --name PyBitmapFontGenerator main.py

PyBitmapFontGenerator
A program for generating bitmap textures of fonts. The peculiarity is that all characters, regardless of their
size are placed in their own cell. Only font selection from outside is available, text color selection is available,
you can change the matrix parameters and font size.
To work with the text field of symbols (select, copy, paste), use the English keyboard layout.

Use the built-in virtual environment.
Compile the program into 1 exe without python on the system: pyinstaller --onefile --windowed --name PyBitmapFontGenerator main.py
